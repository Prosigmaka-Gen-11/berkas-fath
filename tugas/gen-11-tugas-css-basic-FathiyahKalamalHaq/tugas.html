Display Property

Display Property can change how an element is displayed. HTML elements has a default value. There are two type default element level: inline and block. Block means that the element will take a full width space and always make a newline. Meanwhile the inline element is the opposite of block. Inline elements dont make a newline they continue the line. Block-level elements are:

  • div
  • h1-h6
  • p
  • form
  • header
  • footer
  • section
The Inline Elements:
  1. span
  2. a
  3. img
